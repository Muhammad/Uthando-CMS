Template Package Contents

 

The template package include the following.

 

  1. Joomla template zip file which can be installed automatically via joomla.

  2. Photoshop PSD file/s if supplied.

  3. Install instructions.

 

Installing The Joomla Template


1) Login into the administrator area.
2) Select the menu installers and templates-site menu option.
3) Select the Browse button and choose the template.zip file included in this package
4) Select Upload file & Install.
5) The template will now be installed on your server.
6) You can set the template as the default design by selecting the site menu option, template manager and site templates.
7) Choose the template you have just installed and select the default button.
8) The template will now be the default design for your web site.

 

Template Module Settings

 

The template uses the following module settings.

 

 

1) Left menu uses left module.
2) Login box uses the left module position.
3) user4 uses the search module
4) Top menu uses the user3 module position

5) Newsflash uses the top module position.

6) user 1 can use poll or syndicate or latest news.

 

 

 

Changing the Site title Header Text

 


The design currently uses the <?php echo $mosConfig_sitename; ?> which pulls in the site name from the global configuration settings.

If you would like to manually change the <?php echo $mosConfig_sitename; ?> and replace it with your own site title please complete the following.

1) Open the index.php template file using notepad.exe
2) Find the following <div class="title"><h1><?php echo $mosConfig_sitename; ?></h1></div>
3) Remove the <?php echo $mosConfig_sitename; ?> and add your web site name.
4) Save the file and upload the index.php file to the flower template directory.

Changing the footer
 

As per the terms and conditions you must not remove the copyright information included in the footer.

To change the footer open the template index.php file and scroll down to the bottom of the page here you will see the following

 

Your footer text goes here

 

To add your footer just replace the above with the following

 

<a href="http://www.mywebsitename.com"><span style="text-decoration: none">My
Web Site Name</span></a>
